# 📊 Sales Data Analysis | End-to-End E-commerce Insights (Python & EDA)

🚀 A real-world data analytics project completed, focused on transforming raw e-commerce sales data into meaningful business insights using Python and data analysis techniques.


## 📌 Project Overview

This project performs an end-to-end **Sales Data Analysis** on a real-world e-commerce dataset sourced from Kaggle.

The objective is to clean, analyze, and visualize the data to extract actionable insights that help understand: 
- Product performance 📦  
- Pricing and discount patterns 💰  
- Rating distribution ⭐
- Correlation analysis 



## 🎯 Objectives

- Clean and preprocess raw messy data 🧹  
- Handle missing values and inconsistent formats  
- Perform Exploratory Data Analysis (EDA) 🔍  
- Create meaningful data visualizations 📊  
- Derive business insights from patterns and trends 💡  

---

## 🛠️ Tools & Technologies Used

- Python 🐍  
- Pandas 🐼
- regex  
- Matplotlib 📊  
- Seaborn 🎨  
- Jupyter Notebook 📓  

---

## 📂 Dataset Information

- Source: Kaggle E-commerce Sales Dataset  
- Contains product-level data including:
  - Product name  
  - Price  
  - Discount  
  - Ratings   
  - Category information  

---

## 🧹 Data Cleaning & Preprocessing

- Handled missing values and inconsistent entries  
- Cleaned currency columns (₹, commas, symbols) using regex  
- Converted data types for accurate analysis  
- Removed or corrected invalid records  

---

## 🔍 Exploratory Data Analysis (EDA)

Key analysis performed:
- Distribution of product prices 📉  
- Rating vs price relationship 📊  
- Discount impact on sales 💰  
- Category-wise performance analysis 📦  
- Identification of top-performing products ⭐
- product-based analysis

---

## 📊 Data Visualization

Created visual insights using:
- Bar charts 📊  
- Histograms 📉  
- Scatter plots 🔗   
- Heatmaps 🔥  

These visualizations helped uncover hidden patterns in customer behavior and product performance.
